Para rodar o projeto é necessário ter instalado as bibliotecas básicas  que a documentação do próprio
Three Js sugere (https://threejs.org/docs/#manual/en/introduction/Installation), além do Node:
        - Three
        - Vite
É possível adquiri-lás utilizando 'node install' no terminal, estando na pasta 
em que os itens package.json se encontram.

Uma vez que as dependências estiverem instaladas e
a pasta aonde o projeto se econtra estiver aberta, 
basta utilziar comando para inicializar o servidor: 'npx vite'
